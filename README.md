# Android Text to Image Converter

A powerful Android application written in Java that allows you to easily convert your text into images in various formats. This app supports all Android versions, from the oldest to the latest.

## Features

- **Write text manually or copy & paste**: Easily input your desired text.  
- **Import text from files**: Convert `.txt` files into images.  
- **Convert to multiple image formats**: Supports `PNG`, `JPG`, `JPEG`, `WEBP`, `BMP`, `GIF`.  
- **Custom file naming**: Choose the name of your output image.  
- **Select storage location**: Save images in any folder you want.  
- **Change text and background color**: Fully customize the look of your images.  
- **Adjust text size**: Complete control over the font size.  
- **Support for all Android versions**: Works on both old and new Android devices.

## Technologies & Tools

- Programming Language: **Java**  
- Development Environment: **Android Studio**  
- Advanced Color Picker: **Dialog Color Picker**
